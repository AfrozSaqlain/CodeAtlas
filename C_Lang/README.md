To compile a **C** program using `gcc`, run:

```bash
gcc -o xyz ijk.c
```

where:

- `ijk.c` is the C source file.
- `xyz` is the name of the executable that will be generated.

After a successful compilation, run the executable using:

```bash
./xyz
```

Similarly, to compile a **C++** program, use `g++`:

```bash
g++ -o xyz ijk.cpp
```

where:

- `ijk.cpp` is the C++ source file.
- `xyz` is the name of the executable that will be generated.

Run the executable with:

```bash
./xyz
```
